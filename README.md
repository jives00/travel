# Travel

A personal trip-planning app for organizing trips from idea to itinerary.

## What it does

- **Trips** — create and manage trips, each with its own dates, destinations, and details.
- **Map** — visualize destinations, places, and legs of a trip on a map.
- **Itinerary** — build a day-by-day plan of activities, bookings, and stops.
- **Places** — save and organize places of interest for a trip.
- **Lists** — keep custom lists (packing, prep, to-dos, etc.) tied to a trip.
- **Bookings** — track flights, hotels, and other reservations alongside the itinerary.
- **Weather** — check the forecast for trip destinations.

The app is available as a web app and as an Android app, with offline support on mobile so trip details stay accessible without a connection.
